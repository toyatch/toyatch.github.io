## 2. pyenvインストール

```
cd ~/
git clone https://github.com/pyenv/pyenv
mv pyenv .pyenv

cd ~/.pyenv
src/configure
make -C src

echo 'export PYENV_ROOT="$HOME/.pyenv"'    >> ~/.bash_profile
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bash_profile
echo 'eval "$(pyenv init --path)"'         >> ~/.bash_profile
source ~/.bash_profile
```

## 3. pythonのインストールと設定

```
pyenv install x.x.x
pyenv global x.x.x
python --verison
```
x.x.xには任意のバージョンのPythonを指定
