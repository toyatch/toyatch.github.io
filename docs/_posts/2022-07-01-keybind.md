## keybindメモ

### Base

| key(刻印)    | Base Layer     | Layer3            |
|--------------|----------------|-------------------|
| CapsLock     | 左CTRL (**1)   | 左CTRL            |
| 左CTRL       | CapsLock (**1) | Fn(**2)           |

**1 dipsw2 on
**2 Fn+左Shift長押し後に左CTRL

### Layer3

Fn+Ctrlで設定モード

| key(刻印) | Layer3            |
|-----------|-------------------|
| `         | ESC               |
| Fn+`      | `                 |
| Fn+数字   | F数字             |
| Fn+左SPC  | ALT+` (半角/全角) |
| Fn+右SPC  | ALT+` (半角/全角) |
|           |                   |
| Fn+a      | LEFT              |
| Fn+w      | UP                |
| Fn+d      | RIGHT             |
| Fn+s      | DOWN              |
| Fn+f      | ENTER             |
