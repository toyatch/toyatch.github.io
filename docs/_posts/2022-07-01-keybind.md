## keybindメモ

| key(刻印)    | Base Layer     | Layer3            |
|--------------|----------------|-------------------|
| CapsLock     | 左CTRL (**1)   | 左CTRL            |
| 左CTRL       | CapsLock (**1) | Fn                |

| key(刻印) | Layer3            |
|-----------|-------------------|
| `         | ESC               |
| Fn+`      | `                 |
| Fn+数字   | F数字             |
| Fn+\      | ALT+` (半角/全角) |
|           |                   |
| Fn+a      | LEFT              |
| Fn+w      | UP                |
| Fn+d      | RIGHT             |
| Fn+s      | DOWN              |
| Fn+f      | ENTER             |

**1 dipsw2 on
